# Mental Health Datathon Project

## Overview
This project analyzes mental health trends in the technology industry using survey data. The goal is to explore factors influencing mental health treatment among tech workers.

## Dataset
OSMI Mental Health in Tech Survey

## Workflow
1. Data Understanding
2. Data Cleaning
3. Exploratory Data Analysis
4. Machine Learning Model
5. Model Evaluation

## Model
Logistic Regression

Model Accuracy: **~79%**

## Key Insights
- Mental health frequently interferes with work productivity.
- Family history strongly influences treatment seeking.
- Workplace mental health benefits influence employee support.

## Tools Used
Python, Pandas, Seaborn, Scikit-learn, Google Colab
